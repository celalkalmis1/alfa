## Unit Tests

All unit tests are written using Python's built-in `unittest` module. Each test file targets a specific function from `implementation.py`.

| File | Function Tested | Description |
|------|----------------|-------------|
| `unittest_Diagonal.py` | `diagonal()` | Tests calculation of a rectangle's or polygon's diagonal given its dimensions |
| `unittest_exp.py` | `exp()` / `power()` | Tests exponentiation function for correct results and edge cases |
| `unittest_factorial.py` | `factorial()` | Tests factorial computation including edge cases (0!, negative input) |
| `unittest_fibonacci.py` | `fibonacci()` | Tests Fibonacci sequence generation for correctness and boundary values |
| `unittest_prime` | `is_prime()` | Tests prime number checker for primes, non-primes, and edge cases (0, 1, negatives) |
| `unittest_triangle.py` | `triangle()` | Tests triangle validity checker: verifies if three sides satisfy the triangle inequality |

### Running All Unit Tests

```bash
# Run all tests at once
python -m unittest discover -p "unittest_*.py"

# Run a specific test file
python -m unittest unittest_factorial.py
python -m unittest unittest_fibonacci.py
python -m unittest unittest_prime
```

## Running the Project

### 1. Install Dependencies

```bash
pip install flask
```

### 2. Run the Web Application

```bash
python implementation.py
```

Then open your browser and go to:

```
http://127.0.0.1:5000
```

### 3. Run Unit Tests

```bash
python -m unittest discover -p "unittest_*.py"
```

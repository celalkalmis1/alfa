# alfa

A personal academic repository containing coursework, exercises, and projects completed during the Erasmus+ exchange program at PANS Jarosław (Państwowa Akademia Nauk Stosowanych w Jarosławiu), Jarosław, Poland.

---

## About

All code in this repository was written and organized as part of the university courses attended throughout the Erasmus+ exchange program at PANS Jarosław.

---

## Repository Structure

```
alfa/
├── Se/           # Software Engineering
├── mpl/          # Modern Programming Languages
├── oop/          # Object Oriented Programming
└── grading/      # Exam & grading files
```

---

## Courses

### Se — Software Engineering

Python-based backend logic combined with a static web application (HTML, CSS, JavaScript). Includes a full unit testing suite covering mathematical functions such as factorial, Fibonacci, prime checking, triangle validation, diagonal calculation, and exponentiation.

- **Language:** Python 3, HTML5, CSS3, JavaScript
- **Framework:** Flask (Jinja2 templating)
- **Testing:** Python `unittest`
- [View Se README](./Se/README.md)

---

### mpl — Modern Programming Languages

A collection of Python programs covering fundamental programming concepts including algorithms, data structures, list operations, loops, functions, unit conversions, and data visualization.

- **Language:** Python 3
- **Libraries:** matplotlib
- [View mpl README](./mpl/README.md)

---

### oop — Object Oriented Programming

C# programs demonstrating object-oriented design principles. Includes a network diagnostic tool that scans connected PCs and reports IP addresses, TTL values, and hostnames, as well as a student management system using LINQ queries with file output.

- **Language:** C# (.NET)
- **Concepts:** Classes, inheritance, LINQ, file I/O, network simulation
- [View oop README](./oop/README.md)

---

### grading

Exam questions and answer files covering topics from Se, mpl, and oop courses.

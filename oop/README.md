## Network Diagnostics (comp series)

These programs simulate a **server/client network environment**. The main PC acts as a **server** and scans the network; it discovers connected PCs, retrieves their **IP addresses**, **TTL values**, **hostnames**, and other network output similar to a `ping` or `tracert` result.

| File | Description |
|------|-------------|
| `comp06.cs` | Core network scanner: discovers PCs on the network, prints IP addresses, TTL, hostname, and ping response for each connected client |
| `comp06_new.cs` | Updated version of `comp06.cs` with improved output formatting and additional network fields |
| `comp07.cs` | Extends comp06; adds filtering or sorting of discovered devices by IP range or TTL value |
| `comp08.cs` | Adds error handling for unreachable hosts; logs failed connections separately |
| `comp09.cs` | Introduces OOP structure: `Computer` class with properties `IP`, `TTL`, `Hostname`, `Status` |
| `comp10.cs` | Full object-oriented network report: creates a list of `Computer` objects, iterates and prints a formatted network summary table |
| `deneme.cs` | Test/scratch file used during development of the comp series |
| `deneme.exe` | Compiled executable of `deneme.cs` for quick testing |

## Student Management (linq5 series)

These programs mirror the `comp` network output structure but applied to **student data**. They use **LINQ** queries to filter, sort, and group students, then save results to `.txt` files.

| File | Description |
|------|-------------|
| `linq5.cs` | Reads student list; uses LINQ to query and filter students by grade or name; prints results to console |
| `linq5.exe` | Compiled executable of `linq5.cs` |
| `linq5v2.cs` | Version 2: adds saving output to a `.txt` file (`StudentReport.txt`); includes additional LINQ operations (ordering, grouping) |
| `linq5v2.exe` | Compiled executable of `linq5v2.cs` |
| `lib06.cs` | Library/helper class for student data operations; defines `Student` class with properties `Name`, `ID`, `Grade`, `Average` |
| `lib06_new.cs` | Updated version of `lib06.cs` with additional properties and methods |
| `lib07.cs` | Extended student library; adds methods for calculating class average, highest/lowest grade |
| `lib08.cs` | Final version of student library; includes full CRUD-style operations on student list |
| `StudentReport.txt` | Output file generated by `linq5v2.cs`; contains the formatted student report saved from LINQ query results |
| `metin.txt` | Supplementary text file; used as input or output during student data processing exercises |
| `metin.csv` | CSV version of student data; used for import/export operations in student management programs |

## Math & Algorithms

| File | Description |
|------|-------------|
| `euler_fibonacci.cs` | Computes Fibonacci sequence using Euler's method; prints the series up to N terms |
| `euler_leibnitz_pi.cs` | Approximates ПЂ (Pi) using the **Leibniz formula**: `ПЂ/4 = 1 - 1/3 + 1/5 - 1/7 + ...`; prints approximation at each iteration |
| `finding_pi_euler.cs` | Approximates Pi using Euler's formula; compares convergence speed with the Leibniz method |
| `findingpi_with_functions.cs` | Refactored Pi approximation using dedicated functions/methods for cleaner OOP structure |
| `largest_prime.cs` | Finds the largest prime number below a given threshold using a primality-check method |
| `number_pi.cs` | Demonstrates Pi-related calculations and prints Pi to a specified number of decimal places |
| `palindrome.cs` | Checks whether a given string or number is a palindrome; returns `true` or `false` |
| `thousand_project.cs` | Performs a complex calculation or simulation project using numbers up to 1000 |

## Usage

Compile and run any `.cs` file:

```bash
# Compile
csc comp06.cs -out:comp06.exe

# Run
./comp06.exe
```

Or use the .NET CLI:

```bash
dotnet run --project comp06.cs
```

> `.exe` files included in the repository are pre-compiled and can be run directly on Windows.
